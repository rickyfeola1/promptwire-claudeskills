📁 What's inside

Three prompts.
Zero repeated instructions.

You don't write the Skill yourself. You answer questions and Claude builds it. Here's what each prompt does.

01
Figure out what to build

The Skill Readiness Interview

Describe your three most repeated tasks. Claude scores each one on frequency, instruction specificity, and output consistency — then ranks them and tells you which to build first and exactly how much time you'd save per week.

02
Build the full Skill

The Skill Generator

Answer six questions about your task. Claude generates a complete ready-to-upload Skill file — trigger description, step-by-step workflow, output format, edge cases, and real examples. Paste straight into your settings. Done.

03
Fix what's broken

The Skill Debugger

Skill not firing? Firing on the wrong request? Wrong output? Paste your Skill, describe what happened versus what you expected. This prompt diagnoses the exact failure type, gives you the corrected section, and suggests a test to confirm the fix worked.

🚀 Deployment

Four steps. No code. No files.

Everything happens inside Claude's settings page. No terminal, no developer, nothing outside your browser.

01

Enable code execution

Settings → Capabilities → turn on Code execution and file creation. Skills need this to run.

02

Open the Skills page

Customize → Skills inside your Claude account. This is where all your Skills live.

03

Paste your Skill in

Click Create a new Skill, hit the + icon, paste the output from Prompt 2 directly. No file needed.

04

Test it

Open a new conversation, type one of your trigger phrases. Claude picks it up automatically.
